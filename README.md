# Smart Calc Studio

A sleek and responsive calculator website designed for fast and accurate calculations. Built with a clean user interface and smooth button interactions to provide a modern user experience.

## Features
✨ Simple and user-friendly design  
✨ Responsive layout for all devices  
✨ Smooth button hover effects  
✨ Fast and accurate calculations  
✨ Modern black & white theme  

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Purpose
This project was created to improve frontend development skills and practice real-world UI design concepts using core web technologies.

## Future Improvements
- Calculation history

- Keyboard support
